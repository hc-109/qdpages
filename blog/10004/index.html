    Dupay：一款支持USDT消费的创新预付信用卡

    Dupay（前身为Depay）是一种突破性的虚拟信用卡产品，实质上为预付卡。它支持USDT（TRC20）加密货币充值，并内置英镑、欧元和美元电子账户。这款卡片允许用户自由地在英镑、欧元、美元等法定货币与USDT之间进行交易和转账，有效解决了加密货币与法币之间的资金流通问题。它不仅避免了传统加密货币提现时可能遇到的资金冻结风险，还为用户提供了无国界的电子货币支付体验。

    Depay由华人在英国创立的公司开发，旨在为个人用户提供多场景的加密货币应用解决方案。在中国境内，它支持微信、支付宝和美团等主流支付平台的消费。

    注册流程

    注册Dupay无门槛，现在申请Lite卡甚至可以免实名。只需简单几步即可完成：

    1. 首先，点击此处注册账号，建议使用邮箱注册。
    2. 注册完成后，按提示下载并安装iOS或Android版APP。
    3. 打开APP后，先向USDT钱包地址充值： • 仅支持USDT（TRC20） • 充值金额需超过5USDT才会入账 • 激活开卡需要超过10USD • USDT消耗包括：交易所提币手续费、USDT开卡费、USDT兑USD损耗以及充值USD到卡的手续费 • 综合考虑，建议向Depay钱包充值至少20USDT
    4. 在APP中根据个人需求申请相应等级的信用卡。综合评估后，建议开通高级或白金等级。目前仅支持申请MasterCard。
    5. 如果不选择Lite等级卡，需要完成KYC认证。按照指示一步步操作，通常几分钟内系统就能自动完成审核。
    6. 开卡成功后，需要将钱包中的USDT兑换成美元并充入卡中，才能查看对应的卡号。切记不要向任何人泄露你的卡号，某些场景下可直接使用卡号进行小额消费，无需验证CVV。

    费用明细

    使用Dupay（原Depay）卡时，除了一次性开卡费外，日常使用主要涉及以下费用：

    1. 兑换损耗：将USDT兑换成USD时，实际兑换比例不是1:1。此处损耗约为1%+，即转入100 USDT，实际兑换到卡内的金额为98+ USD。
    2. 充值费：如上表所示，将USD充值到虚拟卡作为消费金额时，需按相应比例支付手续费。
    3. 月费：如上表所示，每月初从虚拟卡余额中扣除相应美元。余额不足会产生欠款，下次有余额时将一次性扣除。官方声明，欠费超过半年将自动注销卡片。

    综上所述，以MasterCard高级等级卡为例，日常USDT使用的综合费率约为3%，在同类产品中属于较低水平。

    常见问题解答（FAQ）

    如何绑定支付宝、美团、微信支付？

    1. 重要提示：先往卡内充值，再进行绑定！这点非常关键，请务必牢记。
    2. 绑定支付宝的流程与添加其他银行卡相同，输入卡号、过期日期（月/年）、CVV即可。
    3. 绑定微信支付的步骤也与添加其他银行卡一致，同样需要输入卡号、过期日期（月/年）、CVV。
    4. 绑定美团支付时，首先需要绑定一张国内银行卡，然后关闭美团急速支付功能。下单付款时，切换支付方式，选择国际卡支付，添加Depay的卡片信息。注意，每次点外卖时都需要输入CVV进行验证。
    5. 账单地址可以填写国内的实际地址。
    6. 如需绑定美区App Store，若遇到困难，可尝试切换至美国IP再试。账单地址最好填写美国地址。如果仍然无法绑定，可以联系苹果官方客服，说明这是一张美国虚拟卡。

    绑定支付宝或微信失败，或使用时遇到问题怎么办？

    1. 如果支付宝绑定或支付失败，请在支付宝界面联系在线人工客服（注意：非人工发送的那个入口通常无法打开）。说明遇到的问题，按照客服的要求操作，上传Depay APP首页截图等资料，通常可以解决。
    2. 微信绑定或支付失败时，需要联系微信客服进行申诉。一般也是上传Depay APP首页截图来解决（请注意在文字说明中提到Depay APP截图上有你的个人拼音名字，有时客服可能会忽视这一点）。如果多次申诉无效，可以持续给予差评，直到有客服主动联系你，或者你也可以拨打微信客服电话95017，清楚地解释这是你本人的卡，上面有你的名字。
    3. 美团支付遇到问题？联系美团客服，可以这样说：“这是我自己的虚拟卡，我的朋友都能正常使用。”
    4. 解决所有风控问题的关键是：明确表示这是虚拟卡，是你本人的卡，不是盗刷，态度要坚定自信。

    为什么会遇到风控，如何避免？

    1. 这是一张海外银行卡，国内支付工具一开始就会将其风险等级提高。
    2. 绑定成功后的第一笔消费不要选择虚拟产品，比如手机话费充值。
    3. 可以选择在大型商家消费，例如使用KFC、肯德基的微信小程序。
    4. 避免一开始就进行大额消费，或者超过卡片余额的消费。
    5. 在支付宝和微信上逐步正常使用，慢慢养卡，单次消费金额会逐渐增加。
    6. 建议先充值再使用，这样可以避免很多麻烦。如果因余额不足触发商家风控，后续即使充值也可能无法使用。

    Dupay(原Depay)日常使用建议

    1. 支付宝：适合线下被扫和主动扫码支付，几乎适用于所有日常线下购物场景（注意避免频繁大额支付）。超过200元需支付3%手续费（由支付宝收取）。
    2. 微信：适合在京东、拼多多等平台购物，无额外手续费。但线下二维码商户支持较少（已测试罗森便利店和永辉超市支持）。
    3. 美团：非常适合点外卖，使用方便。如果提示不支持该银行卡，可以直接选择并输入过期日期和CVV，通常就能支持了。
    4. 京东、拼多多、淘宝在购买某些虚拟商品和海外商品时可能不支持此卡支付。
    5. 在淘宝和拼多多上直接绑卡消费，无论金额大小都会收取3%的手续费。
    6. 请记住，这是一张用于日常消费的卡片。不要频繁尝试套现，这不仅不合理，而且容易触发风控机制。
